# Kasa - Projet 11 OpenClassrooms

Kasa est une application moderne de location de logements développée avec React et Vite. Elle permet aux utilisateurs de parcourir, explorer et réserver des logements tout en offrant une expérience utilisateur fluide et intuitive. Ce projet a été réalisé dans le cadre du parcours Développeur d'application JS-React chez OpenClassrooms.

## Fonctionnalités principales

- **Affichage des logements disponibles** : Parcourez une liste de logements avec des informations détaillées.
- **Détails des logements** : Consultez les descriptions, équipements, et notes des logements.
- **Système de notation** : Visualisez les évaluations des logements avec des étoiles.
- **Affichage des équipements** : Découvrez les équipements inclus dans chaque logement.
- **Bannière dynamique** : Profitez d'une bannière avec images et diaporama.

## Technologies utilisées

- **React** : Bibliothèque JavaScript pour construire des interfaces utilisateur.
- **Vite** : Outil de développement rapide pour les projets modernes.
- **React Router** : Gestionnaire de routage pour les applications React.
- **Sass** : Préprocesseur CSS pour des styles modulaires et maintenables.
- **FontAwesome** : Icônes vectorielles pour enrichir l'interface utilisateur.

## Instructions pour lancer le projet

1. Clonez ce dépôt sur votre machine locale :
   ```bash
   git clone <URL_DU_DEPOT>
   cd Kasa.github.io
   ```
2. Installez les dépendances :
   ```bash
   npm install
   ```
3. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```
4. Accédez au site :
   Ouvrez votre navigateur et allez à l'adresse [http://localhost:3000](http://localhost:3000).
