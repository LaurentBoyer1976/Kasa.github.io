# Kasa - Application de location de logements

Kasa est une application moderne de location de logements développée avec React et Vite. Elle permet aux utilisateurs de parcourir, explorer et réserver des logements tout en offrant une expérience utilisateur fluide et intuitive.

## Fonctionnalités

- **Affichage des logements disponibles** : Parcourez une liste de logements avec des informations détaillées.
- **Détails des logements** : Consultez les descriptions, équipements, et notes des logements.
- **Système de notation** : Visualisez les évaluations des logements avec des étoiles.
- **Affichage des équipements** : Découvrez les équipements inclus dans chaque logement.
- **Bannière dynamique** : Profitez d'une bannière avec images et diaporama.

## Technologies utilisées

- [React](https://reactjs.org/) : Bibliothèque JavaScript pour construire des interfaces utilisateur.
- [Vite](https://vitejs.dev/) : Outil de développement rapide pour les projets modernes.
- [React Router](https://reactrouter.com/) : Gestionnaire de routage pour les applications React.
- [Sass](https://sass-lang.com/) : Préprocesseur CSS pour des styles modulaires et maintenables.
- [FontAwesome](https://fontawesome.com/) : Icônes vectorielles pour enrichir l'interface utilisateur.

## Installation et lancement du site

Pour installer les dépendances et lancer le site en local, suivez les étapes ci-dessous :

1. **Cloner le dépôt** :
    ```bash
    git clone https://github.com/votre-utilisateur/votre-repo.git
    cd votre-repo
    ```

2. **Installer les dépendances** :
    ```bash
    npm install
    ```

3. **Lancer le serveur de développement** :
    ```bash
    npm run dev
    ```

4. **Accéder au site** :
    Ouvrez votre navigateur et allez à l'adresse [http://localhost:3000](http://localhost:3000).

## Déploiement

Pour déployer l'application, vous pouvez utiliser des services comme Vercel, Netlify ou GitHub Pages. Voici les étapes pour un déploiement sur GitHub Pages :

1. **Construire l'application** :
    ```bash
    npm run build
    ```

2. **Déployer sur GitHub Pages** :
    Suivez les instructions de [déploiement sur GitHub Pages](https://vitejs.dev/guide/static-deploy.html#github-pages).
